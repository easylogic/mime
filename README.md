# mime
all in one web component for many mime types 

# sample code 

```html
<!-- color view --> 
<mime value="#ff0000" />

<!-- gradient view --> 
<mime value="linear-gradient(black, white)" />

<!-- youtube view --> 
<mime value="https://youtube.com/xxxxxx" />

<!-- image view --> 
<mime value="img.png" />

<!-- iframe view --> 
<mime value="https://webpage /> 
             
<!-- qrcode -->
<mime type="image/qrcode" value="https://www.easylogic.studio" />
```
